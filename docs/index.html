Cadence

Cadence is a small app for keeping up with people without turning friendship into homework.

I made it for myself and my friends. It reads message metadata from your Mac, helps you notice who you have not talked to in a while, and lets you set a rhythm for staying in touch.

Download for Mac
GitHub

macOS 13+ required.

  1. Download Cadence.zip, move the app to Applications, and grant Full Disk Access so it can read message metadata. It does not read message content.
  2. Open the Mac app and click Pair iPhone.
  3. Install the iPhone app through TestFlight, open it, and allow Contacts access.
  4. Scan the QR code from the Mac while both devices are on the same Wi-Fi.
  5. Set cadences for the people you care about and let the app quietly keep track.
  • Cadence only reads handles and timestamps. Message content is never accessed, stored, or uploaded.
  • If something is broken or confusing, report an issue.
