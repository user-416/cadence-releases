A simple loop.

Cadence is meant to get out of the way. It keeps the setup lightweight and the signal clear.

Step 1

Install on Mac

Download Cadence.dmg, move it to Applications, and grant Full Disk Access so the app can read message metadata.

Step 2

Scan and import

Click Pair iPhone on the Mac. On iPhone, open Cadence and scan the QR code while both devices are on the same Wi-Fi.

Step 3

Follow through

Cadence tracks recency, lets you set a rhythm, and quietly shows who could use a check-in.

Quick start, not a manual.

The setup is short. Most of the work is just granting the right permissions once, then scanning the code.

Private by default Cadence only reads handles and timestamps. Message content is never accessed, stored, or uploaded.
Designed for momentum Scores, reminders, and private notes help you remember what matters without turning friendship into admin.

Mac

  1. Download and open Cadence.dmg.
  2. Move Cadence to Applications.
  3. Open it. If macOS blocks launch, use right-click > Open.
  4. Grant Full Disk Access.
  5. Click Sync Now, then Pair iPhone.

iPhone

  1. Install Cadence through TestFlight.
  2. Allow Contacts access when prompted.
  3. Tap the QR scanner icon.
  4. Scan the code from the Mac app.
  5. Set cadences for the people you want to keep up with.
